# Ringkasan Tugas 4, 5, dan 6

## Tugas 4 – Dasar Python

Di tugas ini belajar dasar Python seperti list, indexing, dan slicing untuk ambil data tertentu.

Kesimpulan:
List itu penting banget karena jadi dasar buat ngolah data di Python.

---

## Tugas 5 – Manipulasi Data

Di tugas ini fokus ke manipulasi data, seperti nambah, ubah, dan hapus isi list.

Kesimpulan:
Manipulasi data bikin kita bisa ngatur data dengan lebih fleksibel sesuai kebutuhan.

---

## Tugas 6 – NumPy & Pandas

Di tugas ini mulai pakai library:

- NumPy untuk bikin data random dan hitung statistik (mean, median, std, min, max)
- Pandas untuk bikin DataFrame
- Ada juga penentuan status lulus berdasarkan nilai (>= 70)

Kesimpulan:
NumPy dan Pandas ngebantu banget buat ngolah dan analisis data jadi lebih cepat dan rapi.
